## TIC TAC TOE - real time online

#### Live: https://tictactoeorigin.web.app/

***

#### How to run dev

Client

```
 cd client
 yarn
 yarn start
```

Server

```
- cd server
- yarn
- yarn dev
```

Go to: http://localhost:3000
